**基础常识**

核心硬件

- 显卡/GPU

  目前主要分为 N 卡和 A 卡，也就是英伟达 NVIDIA 和 AMD。他们两家主要负责出核心芯片，而封装成台式机可用的显卡，还需要集成“显卡 PCB 主板”、“显存”、“供电”、“显卡散热”等内容，这个集成则是类似华硕、微星、技嘉、七彩虹、影驰、索泰、蓝宝石、撼讯等厂商做的事情。所以同一个型号，如 RTX 4060 Ti，也会因为集成做工、品牌、散热、频率、保修不同从而导致价格不同。

  同型号显卡之间，核心性能通常差距不大，主要区别在：

  - 散热规模：双风扇、三风扇，影响温度和噪音。
  - 供电做工：影响稳定性和超频空间。
  - 频率设定：OC 版可能略高。
  - 显卡尺寸：会影响机箱是否装得下。
  - 外观灯效：白色、黑色、RGB 等。
  - 售后保修：不同品牌和渠道会有差异。

  显卡和 CPU 要注意搭配。如果 CPU 很强但显卡一般，很多游戏里会先卡在显卡上，这种情况可以理解为“CPU 富养显卡”。好处是以后升级显卡时 CPU 不容易拖后腿，缺点是当前游戏帧率不会因为 CPU 很强就无限提高。

- CPU

  CPU 主要也是两家，Intel 和 AMD。

  AMD 目前常见的是 Ryzen 系列：

  - Ryzen 5：主流游戏和日常使用。
  - Ryzen 7：更强的游戏和综合性能。
  - Ryzen 9：多核心更强，适合剪辑、编译、渲染、多开等生产力场景。

  Intel 常见的是 i5 / i7 / i9，以及新一代 Core Ultra 系列。

  常见尾缀：

  - F：不带核显，需要独立显卡。
  - X：频率或性能更高。
  - X3D：带 3D V-Cache，大缓存，游戏表现强。
  - K：Intel 可超频型号

  选择 CPU 时要看用途。纯游戏可以更关注单核性能和 X3D；剪辑、编译、虚拟机、多任务则更需要核心线程数。

- 主板

  类似显卡，主板也要区分“平台/芯片组”和“具体厂商型号”。芯片组决定大方向，例如支持什么 CPU、什么内存、多少接口；华硕、微星、技嘉、华擎、铭瑄等厂商再基于这些芯片组做出具体主板。

  AMD 常见主板平台：

  - B550：AM4 老平台，DDR4，适合 Ryzen 3000/5000。
  - B650/B850：AM5 新平台，DDR5，适合 Ryzen 7000/9000。
  - X670/X870：AM5 高端平台，扩展更多，价格更高。

  Intel 常见主板平台：

  - B660/B760：LGA1700，适合 12/13/14 代酷睿。
  - B860：LGA1851，新 Core Ultra 平台。
  - Z 系列：高端平台，通常扩展更多，也支持部分 CPU 超频。

  注意：B550 和 B650/B850 不是同一代，不能混用。比如 Ryzen 7000/9000 是 AM5 平台，不能插 B550。

  选主板主要看：

  - CPU 插槽是否匹配。
  - 支持 DDR4 还是 DDR5。
  - M.2 固态插槽数量和 PCIe 版本。
  - 显卡插槽规格。
  - 供电和散热片。
  - 是否带 WiFi、蓝牙、2.5G 网口。
  - 是否有足够的风扇接口和 5V ARGB 接口。

- 散热器

  注意散热器主要是给 CPU 散热的，机箱风扇是为了加速机箱内部热量扩散到空气里。散热分为风冷和水冷。

  风冷：

  - 结构简单，稳定耐用。
  - 性价比高。
  - 常见有单塔、双塔。
  - 缺点是体积可能比较大，可能挡内存。

  水冷：

  - 常见有 240、280、360 冷排。
  - 颜值高，适合海景房机箱。
  - 360 水冷散热余量更大。
  - 缺点是价格更高，有水泵寿命和漏液风险，低价水冷要注意质保。

  水冷接线一般有几类：

  - 水泵线：接 AIO_PUMP。
  - 冷排风扇线：接 CPU_FAN 或水冷自带集线器。
  - ARGB 灯光线：接 5V ARGB 或灯控集线器。

- 风扇

  机箱风扇主要负责进风和出风，把外部冷空气吸进来，把内部热空气排出去。常见尺寸有 12cm 和 14cm，要看机箱预留的风扇位大小。

  风扇主要注意：

  - 风扇转速控制

    - DC 调速（旧）：3pin 接口，通过调节电压控制转速，控制精度不如 PWM。
    - PWM 调速（新）：4pin 接口，主板通过 PWM 信号精准控制转速，推荐优先选。

  - 风扇灯光控制

    - RGB（旧）：老式灯光接口，一般是 12V 4pin，不能和 5V ARGB 混插。
    - ARGB（新）：灯光控制，一般是 5V 3pin 接口。

  - 集线器

    集线器可以理解为风扇/灯光的“插线板”。主板上的风扇接口和 ARGB 接口数量有限，如果风扇很多，就需要集线器。

    常见两类：

    - 风扇转动集线器（PWM 风扇集线器）：把多个风扇接到一个主板风扇接口上。
    - 风扇灯光集线器（ARGB 灯光集线器）：把多个灯光设备接到一个 5V ARGB 接口上。

    很多集线器还需要接 SATA 供电。这样风扇主要从电源取电，主板只负责控制信号，更稳定。

  PWM 的转速一般由主板控制。可以在 BIOS 或主板软件里设置风扇曲线，比如 CPU 温度越高，风扇转速越高。如果风扇接到集线器，集线器只是把主板的控制信号分发给多把风扇，多把风扇通常会按同一条曲线一起转。

  带灯风扇通常有两根线：

  - 4pin PWM：风扇电机供电 + 转速控制，接 SYS_FAN / CHA_FAN 或集线器。
  - 5V 3pin ARGB：灯珠供电 + 灯效控制，接主板 5V ARGB 或灯控集线器。

  正叶和反叶：

  - 普通正叶风扇：通常从正面进风，从背面/支架面（也就是安装贴附面）出风。
  - 反叶风扇：风向相反，适合海景房底部和侧面进风时保持正面好看。

- 内存

  内存是 CPU 临时处理数据的高速空间。新平台常见 DDR5，老平台常见 DDR4，二者不能混用，要看主板支持哪一种。

  频率和时序：

  - DDR5 6000 是 AM5 平台常见甜点频率。
  - CL28 / CL30 / CL32 / CL36 都可以。
  - CL 数字越低，理论延迟越低，但实际游戏差距不一定大。

  优先买套条，比如 16Gx2 或 32Gx2。不要买单条 32GB 做单通道。也不建议随便买两套 16Gx2 凑四条，DDR5 四条更容易降频或不稳定。想要 64GB，优先买 32Gx2。

  内存装好后，需要在 BIOS 里开启 EXPO/XMP，否则可能默认只跑 4800/5200，不能发挥标称 6000MHz 性能。
  
- 磁盘

  磁盘负责长期存储系统、软件、游戏和文件。现在主盘一般优先选 M.2 NVMe 固态硬盘，机械硬盘更适合做大容量仓库盘。

  - 固态

    固态主要看容量、接口、速度、颗粒、主控、保修。

    常见 M.2 NVMe SSD 使用 PCIe 通道：

    - PCIe 3.0 x4：约 3500MB/s。
    - PCIe 4.0 x4：约 7000MB/s。
    - PCIe 5.0 x4：约 10000-14000MB/s+。

    PCIe 5.0 理论速度更高，但价格更贵、发热更大。游戏和日常使用中，PCIe 4.0 高速盘已经很快，体感差距不大。

    对游戏机来说，2TB PCIe 4.0 往往比 1TB PCIe 5.0 更实用，因为容量更重要。

  - 机械

    机械硬盘容量大、价格低，但速度慢、噪音和震动更明显。适合存照片、视频、备份、冷数据，不建议作为系统盘或游戏主盘。

- 电源

  主要看瓦数，没啥好说的

- 机箱

  机箱决定能不能装下主板、显卡、电源、水冷和风扇，也影响风道和颜值。

  选机箱主要看：

  - 支持 ATX / MATX / ITX 哪种主板尺寸。
  - 显卡限长。
  - CPU 风冷散热器限高。
  - 水冷冷排支持位置和尺寸。
  - 风扇位数量和尺寸。
  - （不那么重要）电源长度和走线空间。
  - （不那么重要）是否自带风扇或集线器。
  
  常见机箱风道：
  
  - 底部：进风，通常选反叶。
  - 侧面：进风，通常选反叶。
  - 顶部：出风，通常用正叶或水冷冷排风扇。
  - 后部：出风，通常用正叶。

插口和协议

- PCIe

  PCIe 全称 PCI Express，是电脑内部高速通信总线。显卡、M.2 SSD、网卡、采集卡都可能使用 PCIe。

  PCIe 4.0 和 PCIe 5.0 是同一种标准的不同版本，版本越新，理论带宽越高。它们通常可以向下兼容：

  - PCIe 5.0 SSD 插在 PCIe 4.0 插槽上，可以用，但按 4.0 速度跑。
  - PCIe 4.0 SSD 插在 PCIe 5.0 插槽上，可以用，但仍按 4.0 速度跑。

  常见通道数：

  - 显卡通常使用 PCIe x16。
  - M.2 NVMe SSD 通常使用 PCIe x4。

  注意：PCIe 5.0 SSD 和“PCIe 5 电源”不是一回事。前者说的是主板/SSD 通信通道，后者通常是商家对新显卡供电接口的简称。

- sata

  SATA 是较老的存储接口，常用于 2.5 英寸 SATA SSD 和机械硬盘。速度远低于 M.2 NVMe SSD，但兼容性好，适合接大容量机械硬盘或旧硬盘。

  SATA 也常出现在电源供电线上。很多风扇集线器、ARGB 控制器会用 SATA 电源线供电。

- hdmi

  HDMI 是显示输出接口，用于连接显示器、电视。显卡上常见 HDMI 和 DP 接口。

  - HDMI：电视和普通显示器常见。
  - DP：高刷新率显示器更常用。

  显示器线一般接在独立显卡的视频输出口上，不要接主板视频口，除非使用 CPU 核显。

- 网络

  - 2.5G 网口：比传统千兆网口更快，但路由器、交换机也要支持。
  - WiFi 6 / 6E / 7：无线网络版本，数字越新理论规格越高，也要路由器支持。
